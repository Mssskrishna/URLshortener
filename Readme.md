## Clone

Install my project using clone

```bash
  git clone https://github.com/Mssskrishna/URLshortener.git
  cd URLshortner
```
## Install 

Install dependancies and Dev-dependancies

```bash
  cd backend
  npm install
```

## See results

```bash
  npm run devStart
```
